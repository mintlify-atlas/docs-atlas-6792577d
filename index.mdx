## Key Features

Network Analysis is a powerful client-side application that processes Excel files directly in your browser—no backend required.

<CardGroup cols={2}>
  <Card title="Client-Side Processing" icon="browser" color="#fcc72c">
    Upload Excel files and process them entirely in your browser with no server required
  </Card>
  <Card title="Four Visualization Types" icon="chart-network" color="#fcc72c">
    Sankey diagrams, force-directed networks, ego networks, and geographic maps
  </Card>
  <Card title="Real-Time Filtering" icon="filter" color="#fcc72c">
    Filter by origin, destination, and custom columns to focus on specific relationships
  </Card>
  <Card title="Geographic Visualization" icon="map" color="#fcc72c">
    Plot connections on interactive maps with coordinate support and cost analysis
  </Card>
</CardGroup>

## Visualization Types

Explore relationships in your data with four powerful visualization modes:

<CardGroup cols={2}>
  <Card title="Sankey Diagram" icon="diagram-project" href="/visualizations/sankey">
    Flow-based visualization showing volume of connections between nodes with filtering and zoom controls
  </Card>
  <Card title="Gravitational Network" icon="circle-nodes" href="/visualizations/network">
    Force-directed graph layout with weight-scaled nodes and edges powered by vis-network
  </Card>
  <Card title="Ego Networks" icon="users" href="/visualizations/ego-networks">
    Compare up to 4 destination networks side-by-side in a quad-panel view
  </Card>
  <Card title="Geographic Map" icon="earth-americas" href="/visualizations/geographic-map">
    Map-based visualization with arcs between locations and optional cost mode analysis
  </Card>
</CardGroup>
